
## 💻 Usage Guide

1. **Registration/Login**
   - Create new account or login to existing one

2. **Disease Detection**
   - Upload plant leaf image
   - Get instant disease detection results
   - View recommended treatments

3. **History & Analytics**
   - Track all previous detections
   - View success rates
   - Analyze disease patterns

4. **Notifications**
   - Receive alerts for new detections
   - Track treatment progress

5. **Feedback**
   - Rate detection accuracy
   - Provide improvement suggestions

## 🌐 Deployment

This app can be deployed to Heroku:
1. Create Procfile in root directory
2. Set up Heroku environment
3. Deploy using Heroku CLI or GitHub integration

## ⚠️ Important Notes

- Ensure model file is present before running
- Configure database URL in production
- Set up proper environment variables
- Handle file uploads securely
